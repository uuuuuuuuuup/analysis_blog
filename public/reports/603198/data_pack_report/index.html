四、MD&A 深度信息#

数据来源：AI-Tools MCP GetFinancialReport + WebSearch（年报/业绩说明会/研报）

4.1 经营回顾#

2024年度：

  • 营业总收入73.44亿元，同比+8.46%；归母净利润25.89亿元，同比+13.45%
  • 中高档白酒收入57.13亿元，同比+13.76%，毛利率81.07%（同比+1.83pct），占白酒收入比例提升至81.59%
  • 洞藏系列持续放量，结构升级趋势延续
  • 但未达成2024年初设定的营收目标

2025年前三季度：

  • 营业总收入45.16亿元，同比-18.09%；归母净利润15.11亿元，同比-24.67%
  • 25Q3营收13.6亿元，同比-20.8%；归母净利润3.8亿元，同比-39.0%
  • 业绩持续承压，省内外市场双双下滑

业绩下滑原因（管理层说明）：

  1. 经济增速下降，消费需求疲软，消费力不足
  2. 公司主动控量稳价，减轻渠道压力
  3. 行业深度调整期，白酒消费场景收缩

4.2 前瞻指引#

2025年度预算目标：

  • 营业收入76.00亿元，预计同比增长3.49%
  • 净利润26.20亿元，预计同比增长1%
  • 白酒主业目标72.56亿元，预计同比增长3.46%

实际进展： 2025年前三季度营收仅45.16亿元（同比-18.09%），距离全年76亿目标差距巨大，年度目标恐难达成。

长期战略规划：

  • 以"酒、水、石斛、旅游"四大产业为基础，瞄准150亿营收战略目标
  • 深耕安徽市场，突破省外市场，积极拓展电商业务
  • 强化洞藏系列品牌力，推进产品结构升级

4.3 资本配置意图#

  1. 产能扩张： 在建工程从2024年末3.00亿增至2025Q3的8.39亿，长期借款大幅增加，主要用于固定资产投资（酿酒产能扩建）
  2. 委托理财： 公司及子公司拟使用额度不超过60亿元闲置自有资金进行委托理财（银行理财产品、券商理财产品等）
  3. 分红回报： 2024年度拟10派15元，合计分红12亿元，占净利润46.35%
  4. 资产处置： 2025年8月出售全资子公司迎驾商务酒店股权（关联交易）

4.4 风险因素#

  1. 宏观经济风险： 经济增速下降导致消费需求疲软
  2. 行业竞争风险： 白酒行业深度调整，区域酒企面临名酒下沉压力
  3. 库存风险： 存货占流动资产超56%，库存周转压力加大（审计报告将存货列为关键审计事项）
  4. 省外拓展风险： 省外市场突破进展不及预期
  5. 渠道风险： 合同负债同比下滑约11%，渠道信心待修复
