### 任务 3.1：引入中文衬线字体
- **优先级**：P1
- **状态**：待开始
- **描述**：优化网站字体配置
- **具体改动**：

```css
  --font-heading: 'Noto Sans SC', 'Source Han Sans SC', 'PingFang SC', sans-serif;

}

  font-family: var(--font-body);

  line-height: var(--line-height-body);

  max-width: var(--content-width);
  margin: 0 auto;

  margin-bottom: 1.2em;
  text-indent: 0;
}

}
```

## 第四阶段：搜索功能（P1 - 本周内）

### 任务 4.1：启用 Fuse.js 搜索
- **优先级**：P1
- **状态**：待开始
- **描述**：配置Hugo输出JSON索引，启用PaperMod内置搜索
- **具体改动**：
  - 配置outputs支持JSON
  - 启用ShowSearch参数
  - 配置fuseOpts选项
- **预期效果**：用户可快速检索40+报告，Terance无此功能
- **代码位置**：`config.yml`

```yaml
# config.yml
outputs:
  home:
    - HTML
    - RSS
    - JSON  # 必须添加，为搜索提供索引

params:
  ShowSearch: true
  fuseOpts:
    isCaseSensitive: false
    shouldSort: true
    threshold: 0.4
    minMatchCharLength: 2
    keys: ["title", "permalink", "summary", "content"]
```

- **具体改动**：实现类似Terance的「核心指标速览」效果
- **预期效果**：报告首页信息密度高，专业感强
- **代码位置**：
  - `layouts/shortcodes/metrics.html`
  - `assets/css/extended/custom.css`

```html
<!-- layouts/shortcodes/metrics.html -->
<div class="metrics-grid">
  {{ range .Params }}
  <div class="metric-card {{ .class }}">
    <div class="metric-label">{{ .label }}</div>
    <div class="metric-value">{{ .value }}</div>
    {{ if .unit }}
      <div class="metric-unit">{{ .unit }}</div>
    {{ end }}
    {{ if .note }}
      <div class="metric-note">{{ .note }}</div>
    {{ end }}
  </div>
  {{ end }}
</div>
```

```css
.metrics-grid {
```

```markdown
  {"label": "门槛 II", "value": "3.82%", "unit": "A股 max(3.5%, Rf+2%)"}
```
