### 步骤 8：粗算穿透回报率

```
M（支付率锚定值）= 40%
O（年均回购金额）≈ 0.5 亿元
Q（综合股息税率）= 0%

R = [C × M × (1 − Q%) + O] / 当前市值
R = [5.01 × 0.40 × 1.0 + 0.5] / 185.04
R = [2.004 + 0.5] / 185.04
R = 2.504 / 185.04
R = 1.35%
```

### 粗算否决门检验

| 检查 | 条件 | 结果 | 判断 |
|------|------|------|------|
| 主轨 | R < Rf? | 1.35% < 1.78% | ⚠️ 是 |
| EV口径 | R_EV < Rf? | 1.38% < 1.78% | ⚠️ 是 |
| 主轨 | R < II×0.5? | 1.35% < 1.89% | ⚠️ 是 |
| EV口径 | R_EV < II×0.5? | 1.38% < 1.89% | ⚠️ 是 |

> EV = 市值 − 广义净现金 = 185.04 − 3.72 = 181.32 亿元
> R_EV = 2.504 / 181.32 = 1.38%

**粗算结论：否决** — 穿透回报率 R = 1.35%，远低于A股门槛 II = 3.78%，即使使用EV口径（1.38%）仍不通过。
