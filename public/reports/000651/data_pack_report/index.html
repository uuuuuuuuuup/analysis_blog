10.4 数据可靠性说明#

  1. AI-Tools MCP数据：合并口径报表数据来源于东方财富数据库，数据可靠性高，与公司公告一致。
  2. Web Search数据：审计意见、股息分配等来源于公司公告/权威媒体，可靠性高；有息负债拆解来源于雪球用户分析，可靠性中等，需与年报核对。
  3. 推算数据：非经常性损益合计数通过净利润-扣非净利润推算，可能存在四舍五入差异；定期存款通过货币资金-现金等价物推算，仅为近似值。
  4. 2025全年数据：部分来源于公开年报摘要及研报解读，最终以公司正式公告为准。
