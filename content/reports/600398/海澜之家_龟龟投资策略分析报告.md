## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤 1：归母净利润**
- A（集团净利润）= 21.96 亿元
- B（少数股东损益）= 0.30 亿元
- C（归母净利润）= A − B = **21.66 亿元**

**步骤 2：Owner Earnings 粗算**
- D（折旧摊销）= 2.8 亿元
- G（维持性资本开支系数）= 0.5（零售业典型值）
- H（维持性资本开支）= D × G = 2.8 × 0.5 = 1.4 亿元
- I（Owner Earnings）= C + D − H = 21.66 + 2.8 − 1.4 = **23.06 亿元**

**步骤 3：分配能力验证**
- OCF（2025）= 44.82 亿元 > 股息 19.70 亿元 ✅
- OCF（2024）= 23.17 亿元 > 股息 19.69 亿元 ✅（勉强通过）
- **结论：分配能力验证通过**

**步骤 4：现金上游障碍**
- A股上市公司，经营主体均在国内
- 无跨境分红障碍
- **结论：无上游障碍**

**步骤 5：分配意愿**
- 3年平均支付率 = 91.7%
- 连续多年维持90%+高分红
- 上市以来累计分红205.39亿元
- **结论：分配意愿 = 强**

**步骤 6：可预测性**
- 营收增长温和（-2.65%~+3.19%），波动较小
- 利润受减值影响波动较大
- **结论：可预测性 = 中等**

**步骤 7：税务泄漏修正**
- A股持股>1年：股息税 = 0%
- Q = 0%

**步骤 8：粗算穿透回报率**

```
R = [C × M × (1 − Q%) + O] / 当前市值
R = [21.66 × 0.91 × 1.0 + 0] / 325.6
R = 19.71 / 325.6
R = 6.05%
```

### 粗算否决门判断

| 检查项 | 结果 |
|--------|------|
| R < Rf? | 6.05% > 1.78% → 否 |
| R < II × 0.5? | 6.05% > 1.89% → 否 |
| II × 0.5 ≤ R < II? | 6.05% > 3.78% → 否，R > II |

**因子 2 结论：✅ 通过**，粗算穿透回报率 R = 6.05%，显著高于门槛 II = 3.78%
