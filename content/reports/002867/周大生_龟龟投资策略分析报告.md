**评级：低**

- 维持性资本开支系数 G ≈ 0.3-0.5

### 模块二：收款模式

**评级：现结为主，少量应收**

- 应收账款周转天数：23.2天（2024FY），13.4天（2025Q3）

**技术性护城河：窄**

- 黄金珠宝行业产品同质化程度高，技术壁垒有限

**综合护城河评级：窄**

- 品牌和渠道提供一定护城河，但行业竞争激烈
- 金价透明度高，产品差异化有限

## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤 1：归母净利润**
C = 10.02 亿元

**步骤 2：Owner Earnings 粗算**
- D（折旧摊销）≈ 1.50 亿元
- E（资本开支）= 2.05 亿元
- G（维持性资本开支系数）= 0.5（珠宝零售轻资产，维持性开支低）
- H = D × G = 1.50 × 0.5 = 0.75 亿元
- I = C + D - H = 10.02 + 1.50 - 0.75 = 10.77 亿元

> 保守口径：I = C + D - E = 10.02 + 1.50 - 2.05 = 9.47 亿元

**步骤 3：分配能力验证**
- OCF = 18.56 亿元 > 归母净利润 = 10.02 亿元 ✅
- OCF/净利润 = 1.85x，现金生成能力极强
- **一票否决检查：通过**

**步骤 4：现金上游障碍**
- A股公司，无跨境分红障碍
- 少数股东权益为负，无少数股东分流

**步骤 5：分配意愿**
- 3年支付率序列：60% → 78% → 103%，持续提升
- 2024年支付率103%，分红超过净利润
- 上市以来持续分红，未中断
- **分配意愿：强**

**步骤 6：可预测性**
- 营收受金价影响波动较大（2024年-14.73%，2025Q3 -37.35%）
- 但净利率改善趋势明确（7.21% → 12.99%）
- **可预测性：中等**

**步骤 7：税务泄漏修正**
- A股长期持有（>1年）：Q = 0%
- 短期持有（1月-1年）：Q = 10%
- 本分析采用长期持有视角：Q = 0%

**步骤 8：粗算穿透回报率**

```
R = [I × M × (1 - Q%) + O] / 当前市值
  = [9.47 × 0.78 × 1.0 + 0] / 141.65
  = 7.39 / 141.65
  = 5.22%
```

**粗算否决门检查：**
- R = 5.22% > Rf = 1.78% ✅
- R = 5.22% > II × 0.5 = 1.89% ✅
- R = 5.22% > II = 3.78% ✅

**粗算结论：✅ 通过，R = 5.22% > II = 3.78%**
