## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1-2：归母净利润与 Owner Earnings

```
C（归母净利润）= 25.32 亿元（2024年度）
D（折旧摊销）= 3.00 亿元
E（资本开支）= 5.50 亿元
G（维持性资本开支系数）= 0.55（中药企业，约55%的capex为维持性）
H（维持性资本开支粗估）= E × G = 5.50 × 0.55 = 3.025 亿元
I（Owner Earnings）= C + D - H = 25.32 + 3.00 - 3.025 = 25.30 亿元

保守修正：OCF = 21.76 亿元 < Owner Earnings = 25.30 亿元
→ 说明营运资本在消耗现金（应收账款增加等）
→ 取保守值：Owner Earnings = OCF - 维持性Capex = 21.76 - 3.025 = 18.74 亿元
```

### 步骤 3：分配能力验证

```
OCF = 21.76 亿元
年度股息 = 19.21 亿元
OCF - 股息 = 2.55 亿元 > 0 → 勉强通过

但 Owner Earnings（18.74）< 股息（19.21）→ ⚠️ 分配超过盈利能力
→ 2024年75.90%的高支付率在利润下滑背景下不可持续
```

### 步骤 4-5：现金上游障碍与分配意愿

- 现金上游障碍：无（A股直接持有，无跨境障碍）
- 分配意愿：**极强**（支付率持续提升，从29%→76%）

### 步骤 6：可预测性

- 营收波动：2020-2023年持续增长，2024年转跌-17%，2025年加速下跌-32%
- 可预测性：**低**（业绩拐点不确定，BD转化需时间）

### 步骤 7：税务泄漏修正

```
A股直接持有，长期持股（>1年）股息免征个人所得税
Q = 0%
```

### 步骤 8：粗算穿透回报率

```
M（支付率锚定值）= 55%（保守取值）
O（年均回购）= 0.25 亿元
Q = 0%

R = [I × M × (1 - Q%) + O] / 当前市值
  = [18.74 × 0.55 × 1.0 + 0.25] / 233.01
  = [10.307 + 0.25] / 233.01
  = 10.557 / 233.01
  = 4.53%
```

### 粗算否决门判断

```
R（4.53%）> Rf（1.78%）→ 通过
R（4.53%）> II × 0.5（1.89%）→ 通过
R（4.53%）> II（3.78%）→ 通过（边际）

⚠️ 但需注意：R 基于2024年数据计算，2025年业绩大幅下滑
若使用2025年预估数据，R 将显著低于门槛
```

**因子 2 结论：边际通过（基于2024年数据），但2025年趋势恶化需警惕**

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1：真实现金收入还原

```
S（营业收入）= 80.17 亿元（2024）
T（应收账款净变动）= 2024年末AR 21.73 - 2023年末AR（估算~24.0）= -2.27 亿元（减少）
  → 应收减少意味着回款好于收入确认
U（预收款/合同负债净变动）= 估算~0 亿元（变动不大）

保守处理：
- 应收净减少 → 加回（现金收入>账面收入）
- 预收款变动 → 不加回

真实现金收入 = S - T_净增加 - U_净减少
             = 80.17 - (-2.27) - 0
             = 82.44 亿元

收款比率 = 82.44 / 80.17 = 102.8% → 优秀
```
