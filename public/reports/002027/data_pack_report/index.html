四、MD&A 深度信息#

数据来源：AI-Tools MCP GetFinancialReport + WebSearch公开信息

4.1 经营回顾（2024FY + 2025前三季度）#

2024年全年业绩：

  • 营业收入122.62亿元，同比+3.01%
  • 归母净利润51.55亿元，同比+6.80%
  • 扣非净利润46.68亿元，同比+6.73%
  • 经营现金流66.42亿元，自由现金流约37.25亿元
  • 毛利率66.26%，净利率41.36%，ROE 29.91%

2025年前三季度业绩：

  • 营业收入96.07亿元，同比+3.73%
  • 归母净利润42.40亿元，同比+6.87%
  • 扣非净利润40.05亿元，同比+13.11%
  • Q3单季毛利率达74.1%，同比提升5.91个百分点，创2018年以来新高
  • 收入结构呈现"消费为基、互联网提速、多点支撑"特征

关键经营亮点：

  1. 设备折旧费同比大幅下降52%（减少约1.69亿元），因2018年扩张的电梯媒体固定资产折旧完毕
  2. 互联网板块收入大幅增长，2025H1收入占比显著提升
  3. 收购新潮传媒（2024年新潮净利润0.33亿元，2023年亏损1.53亿元），行业格局进一步优化
  4. 已签订但尚未履行确认的合同金额约38.1亿元，其中37.3亿元将在2025年确认

4.2 前瞻指引#

  1. 下沉市场红利：三线及以下城市电梯媒体覆盖率仅35%，预计未来3年设备量年均增速超20%，带动区域收入占比突破15%
  2. 出海第二曲线：一带一路沿线国家布局加速，2024年海外营收占比提升至5.2%，东南亚数字广告市场年复合增速18%
  3. 技术驱动：深化AI应用，探索智能投放、效果监测与数据回流闭环，提升广告转化率
  4. “碰一碰"新业务：与支付宝合作推出"碰一下"互动广告，有望开启成长新周期

4.3 资本配置意图#

  1. 高分红策略：2024年度拟10派2.3元，分红率约92.45%，持续维持高分红政策
  2. 2025年中期分红规划：已实施10派1元（2025年8月）和10派0.5元（2025年11月）
  3. 理财产品投资：公司及子公司拟使用不超过80亿元自有闲置资金购买理财产品（其中70亿元用于低风险产品，10亿元用于中等风险产品）
  4. 收购整合：完成新潮传媒收购，行业集中度进一步提升

4.4 风险因素#

  1. 中国广告市场需求不确定风险：宏观经济波动影响广告主投放意愿
  2. 应收账款坏账风险：应收账款余额较年初增加5.76亿元，部分客户回款能力下降
  3. 市场竞争风险：尽管收购新潮，仍需关注其他媒体形式竞争
  4. 媒体资源成本上升风险：优质点位租金可能持续上涨
  5. 海外经营风险：海外扩张面临政策、汇率等不确定性
