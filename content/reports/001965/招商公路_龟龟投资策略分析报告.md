## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤 1：归母净利润**
C = 46.10 亿元

**步骤 2：Owner Earnings 粗算**
D（折旧摊销）≈ 30.00 亿元
G（维持性资本开支系数）= 0.35（高速公路维护开支占比低）
H = D × G = 30.00 × 0.35 = 10.50 亿元
I = C + D - H = 46.10 + 30.00 - 10.50 = 65.60 亿元

**步骤 3：分配能力验证**
OCF（2025）= 65.10 亿元 > 股息支付 25.35 亿元 ✅
FCF（2025）≈ 45.10 亿元 > 股息支付 25.35 亿元 ✅
→ 分配能力验证通过

**步骤 4：现金上游障碍**
A股直接持有，无跨境障碍 ✅

**步骤 5：分配意愿**
支付率均值 ≈ 54%，连续高分红，2024-2025年实施回购 → 分配意愿强

**步骤 6：可预测性**
高速公路通行费收入可预测性高，但投资收益波动带来不确定性 → 可预测性中

**步骤 7：税务泄漏修正**
A股持股>1年：Q = 0%（免税）

**步骤 8：粗算穿透回报率**
R = [I × M × (1-Q%) + O] / 市值
= [65.60 × 0.5498 × 1 + 1.55] / 621.8
= [36.07 + 1.55] / 621.8
= 37.62 / 621.8
= **6.05%**

### 粗算否决门判断

| 条件 | 结果 |
|------|------|
| R < Rf (1.78%)? | 否 ✅ |
| R < II×0.5 (1.89%)? | 否 ✅ |
| II×0.5 ≤ R < II? | 否（R=6.05% > II=3.78%）✅ |

**粗算结论：✅ 通过** — R=6.05%显著高于门槛II=3.78%
