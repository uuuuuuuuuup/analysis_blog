### 模块一：资本消耗强度

**评级：中低**

- 白酒行业本身资本消耗强度低（轻资产模式），但古井贡酒正处于产能扩张期
- 2024FY在建工程10.39亿元（智能园项目），资本开支占营收比约4.2%
- 行业对比：茅台capex/营收约2%，古井偏高但属于成长期特征
- **关键判断**：当前高capex是阶段性的，长期看资本消耗将回归行业低位

### 模块二：收款模式

**评级：预收为主（优秀）**

- 应收账款周转天数仅1.06天（2024FY），几乎可以忽略
- 白酒行业典型的"先款后货"模式，经销商需预付货款
- 合同负债（预收款）约30亿元，体现强势渠道地位
- 收款比率 ≈ 100%，现金回收质量极高
- **关键判断**：收款模式是古井贡酒最大的财务优势之一

### 模块三：竞争格局与护城河

**第一层（非技术性护城河）：窄到中**

| 护城河维度 | 评估 | 说明 |
|-----------|------|------|
| 品牌力 | 中 | 徽酒龙头，安徽市占率>30%，但全国品牌力弱于茅台/五粮液 |
| 网络效应 | 弱 | 白酒无显著网络效应 |
| 转换成本 | 中 | 宴席渠道粘性较高，消费者口味习惯有一定锁定 |
| 规模经济 | 中 | 产能规模优势，但销售费用率26%反映品牌力差距 |
| 特许经营 | 弱 | 无特殊牌照 |

**第二层（技术性护城河）：窄**

- 2025年品牌价值全球烈酒第八、中国白酒第四，但同比下降19.7%
- "年份原浆"概念存在争议，高端化认可度弱于茅台/五粮液/国窖
- 省外收入占比~40-46%但增速放缓，全国化尚未完成
- **关键判断**：区域龙头护城河稳固，但全国化护城河尚未建立；品牌价值下滑是警示信号

### 模块四：周期性

**评级：弱周期**

- 白酒消费有一定周期性（与经济景气度、政务/商务消费相关）
- 2025年行业进入深度调整期，公司Q3营收同比-51.7%
- 但白酒行业不存在典型的产能周期，更多是库存周期
- 古井贡酒产品结构偏次高端（100-500元），受消费降级影响较大
- **关键判断**：当前处于行业调整期底部区域，非周期顶部

### 模块五：人力资本依赖

**评级：低**

- 白酒生产高度依赖酿造工艺和窖池，而非个别人才
- 管理层为国企体系，稳定性高但激励可能不足
- 销售团队规模大但可替代性强
- **关键判断**：人力资本依赖度低，不构成重大风险

### 模块六：管理层与治理结构

**评级：良好**

| 维度 | 评估 | 说明 |
|------|------|------|
| 董事长 | 梁金辉，资深行业人士 | 中国酿酒大师，全国人大代表，行业经验丰富 |
| 控股股东 | 亳州市国资委 | 国企背景，治理规范，无掏空风险 |
| 分红意愿 | 持续提升 | 支付率从36%（2019）提升至57%（2024），机构预期70%+ |
| 关联交易 | 可控 | 2024年实际4,901万元，占营收0.2%，规模小 |
| 审计稳定性 | 优秀 | 容诚所连续7年，标准无保留意见 |
| 资本配置 | 中性偏积极 | 智能园建设+49亿国债逆回购，现金利用效率一般 |

- **关键判断**：国企治理规范，分红意愿持续提升是积极信号；但现金利用效率有优化空间

### 模块七：监管与政策风险

**评级：中**

- 消费税改革预期：白酒行业可能面临税制调整，增加税负
- 限制"三公消费"政策：历史上有过政策冲击
- 食品安全监管：行业共性风险
- 环保监管趋严：对酿酒企业提出更高要求
- **关键判断**：消费税改革是最大政策风险，但短期落地概率较低

### 模块八：MD&A 解读

**评级：中性偏负**

- **经营回顾**：2024年营收+16.41%，归母+20.22%，表现优异；但2025年前三季度营收-13.87%，Q3单季-51.7%
- **前瞻指引**：公司不再提300亿营收目标，"降速发展成为行业共识"；机构预期2025年净利润约43-45亿元（同比-19%）
- **风险因素**：渠道去库存压力、合同负债同比-61.76%、经销商订货意愿低迷
- **积极信号**：主动调整渠道库存是健康举措，为后续恢复奠定基础

### 模块九：控股折价分析

## 【因子 2：穿透回报率粗算（Top-Down）】

### 步骤 1：归母净利润

C = 55.17 亿元（2024FY）

### 步骤 2：Owner Earnings 粗算

```
D = 折旧摊销 ≈ 5.0 亿元（基于固定资产78.97亿 × ~6%折旧率估算）
G = 维持性资本开支系数 = 0.25（白酒行业低维持性capex特征）
H = D × G = 5.0 × 0.25 = 1.25 亿元
I = C + D - H = 55.17 + 5.0 - 1.25 = 58.92 亿元（Owner Earnings）
```

### 步骤 3：分配能力验证

- OCF = 47.28 亿元 > 0 ✓
- FCF = ~37 亿元 > 0 ✓

- **上游障碍极小**

- 机构预期2025年支付率提升至70%+

- **分配意愿积极，信号正面**

- 白酒行业需求相对稳定，但次高端价格带受经济周期影响
- 2025年业绩大幅波动（Q3 -51.7%），短期可预测性下降

- **可预测性：中等**

### 步骤 7：税务泄漏修正

- A股持股>1年：股息免征 → Q = 0%

- **保守取 Q = 0%（假设长期持有）**

```
R = [I × M × (1 - Q%) + O] / 当前市值
```
