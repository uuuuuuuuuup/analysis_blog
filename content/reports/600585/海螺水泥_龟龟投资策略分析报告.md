## 【因子 2：穿透回报率粗算（Top-Down）】

### 步骤1：归母净利润

C = 81.13 亿元（2025FY）

### 步骤2：Owner Earnings 粗算

```
D（折旧摊销）= ~50.00 亿元

H = D × G = 50.00 × 0.6 = 30.00 亿元（维持性资本开支粗估）
I = C + D − H = 81.13 + 50.00 − 30.00 = 101.13 亿元（Owner Earnings）
```


### 步骤8：粗算穿透回报率 R%

```
M = 51.0%（支付率锚定值）
O = 2.50 亿元（年均回购金额）

R = [C × M × (1 − Q%) + O] / 当前市值
  = [81.13 × 51.0% × 1 + 2.50] / 1,195
  = [41.38 + 2.50] / 1,195
  = 43.88 / 1,195
  = 3.67%
```

### 粗算否决门判断

```
Rf = 1.78%
A股门槛 II = max(3.5%, 1.78% + 2%) = 3.78%

R = 3.67%
II × 0.5 = 1.89%

II × 0.5 ≤ R < II → 标注"边际不达标"，继续进入因子3
```

**粗算结论：⚠️ 边际不达标** — R=3.67%略低于门槛3.78%，差距仅0.11pct，进入因子3精算。

## 【因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计】

### 步骤1：真实现金收入还原

```
S（营业收入）= 825.32 亿元
T（应收账款净变动）= AR_2025 − AR_2024 ≈ 30.00 − 37.73 = −7.73 亿元（减少）
U（预收款/合同负债净变动）≈ 0（水泥行业预收占比小）

保守处理：
- 应收净减少7.73亿 → 现金回收超过收入确认 → 加回

真实现金收入 = S − T净增加 − U净减少
             = 825.32 − (−7.73) − 0
             = 833.05 亿元

收款比率 = 833.05 / 825.32 = 100.9% → 优秀
```

- 应收周转天数从16.16天缩短至14.21天
- 一年以上应收占比低（水泥行业特性）
- **无重大应收风险**
