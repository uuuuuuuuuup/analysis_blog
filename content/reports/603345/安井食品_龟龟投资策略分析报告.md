## 因子 2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

C = 13.59 亿元（2025A）

### 步骤2：Owner Earnings 粗算

```
D（折旧摊销）= 4.50 亿元
G（维持性资本开支系数）= 0.50（食品制造业典型值）
H = D × G = 4.50 × 0.50 = 2.25 亿元
I（Owner Earnings）= C + D - H = 13.59 + 4.50 - 2.25 = 15.84 亿元
```

### 步骤3：分配能力验证

| 检查项 | 结果 |
|--------|------|
| OCF vs Owner Earnings | OCF 23.17 > I 15.84 ✅ |
| OCF - Capex vs I | 23.17 - 10.0 = 13.17 < I 15.84 ⚠️ |
| 分配能力判定 | **通过**（OCF足以覆盖Owner Earnings，但FCF偏低需关注） |

### 步骤4：现金上游障碍

A股公司，全资/控股子公司为主，现金上游障碍低。

### 步骤7：税务泄漏修正

A股持股>1年免税：Q% = 0%

### 步骤8：粗算穿透回报率

```
R = [I × M × (1 - Q%) + O] / 市值
  = [15.84 × 70% × 100% + 0.50] / 315.7
  = [11.09 + 0.50] / 315.7
  = 11.59 / 315.7
  = 3.67%
```

### 粗算否决门检查

| 检查项 | 结果 |
|--------|------|
| R = 3.67% vs Rf = 1.78% | R > Rf ✅ |
| R = 3.67% vs II × 0.5 = 1.89% | R > 1.89% ✅ |
| R = 3.67% vs II = 3.78% | R < II ⚠️ |

**粗算结论：边际不达标** — R略低于门槛II，继续进入因子3精算

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

```

U（预收款/合同负债净变动）≈ 0 亿元（占比较低）

保守处理：
- 应收账款净增加 → 扣除
- 预收款净变动 → 不加回

真实现金收入 = S - T = 161.93 - 1.24 = 160.69 亿元
收款比率 = 160.69 / 161.93 = 99.2%
```
