### 步骤 1：计算 Owner Earnings

{{< collapse title="📊 展开计算过程" >}}
```
Owner Earnings = 归母净利润 + 折旧摊销 - 资本开支

- 归母净利润 C = 154.87 亿元

计算：

```

### 步骤 1：计算可支配结余

{{< collapse title="📊 展开计算过程" >}}
```
可支配结余 = OCF - 维持性CAPEX

参数：
- 经营现金流 OCF = 300.56 亿元
- 资本开支 E = 95.29 亿元
- 维持性CAPEX系数 G = 0.7（养殖行业）
- 维持性CAPEX = E × G = 95.29 × 0.7 = 66.70 亿元

计算：
可支配结余 = 300.56 - 66.70
           = 233.86 亿元

考虑股息支付后的可支配结余：
可支配结余（调整后）= 233.86 - 74.38（股息）+ 21.60（回购）
                    = 181.08 亿元

但更准确的计算方式：
可支配结余 = OCF - CAPEX + 净现金变化
          = 300.56 - 95.29 + 0
          = 205.27 亿元
```
{{< /collapse >}}

### 步骤 2：计算精算穿透回报率 GG

{{< collapse title="📊 展开计算过程" >}}
```
精算穿透回报率 GG = 可支配结余 / 当前市值

参数：
- 可支配结余 = 205.27 亿元
- 当前市值 = 2,533 亿元

计算：
GG = 205.27 / 2,533
   = 8.11%
```
{{< /collapse >}}
