## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

- 2024FY GAAP 归母净利润 C = 84.53 亿元
- 2024FY 包含一次性卖矿收益 ~25 亿元
- 2024FY 包含澳优商誉减值 -30.38 亿元
- **调整后归母净利润 C_adj = 84.53 - 25 + 30.38 = 89.91 亿元**
- 2025Q3 年化归母净利润 ≈ 110 亿元（考虑Q4季节性调整后）
- **锚定口径取值：C = ~105 亿元**（C_adj与2025年化值的中间值）

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）≈ 32 亿元
E（资本开支）= 22.23 亿元
G（维持性资本开支系数）= 0.6（乳制品行业中等水平）
H = D × G = 32 × 0.6 = 19.2 亿元
I = C + D - H = 105 + 32 - 19.2 = 117.8 亿元
```

**Owner Earnings = 117.8 亿元**

### 步骤 3：分配能力验证

- OCF = 217.40 亿元（2024FY）>> 股息 77.26 亿元 → ✅ 分配能力充足
- FCF = ~195 亿元 >> 股息 + 回购 = 87.33 亿元 → ✅ 覆盖倍数 2.2x

- 少数股东损益仅0.11亿元，占比极低 → ✅ 无重大泄漏

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1：真实现金收入还原

```
S（营业收入）= 1,157.80 亿元
T（应收账款净变动）= 28.78 - 29.59 = -0.81 亿元（减少，正向）
U（预收款/合同负债净变动）≈ 0（未单独披露，假设微小）

真实现金收入 = S - T_增加 - U_减少
             = 1,157.80 - 0 - 0
             = 1,157.80 亿元

收款比率 = 1,157.80 / 1,157.80 = 100%
```

**收款比率 = 100% → 优秀，无收入虚增风险**

- **结论：应收账款风险极低，无需深度核查**

### 步骤 3：非经常性现金流入分类

### 步骤 4：经营性现金支出还原

⚠️ 现金流量表经营活动流出明细（购买商品/支付税费/支付利息）API未返回分项数据。

采用 **OCF 倒推法**：

```
W = 真实现金收入 + V1 + V5 - OCF
  = 1,157.80 + 25 + 1.74 - 217.40
  = 966.14 亿元

标注：⚠️ 现金支出明细不足，已采用 OCF 倒推法估算 W
```

**注意**：卖矿收入的现金在投资CF中体现，不在经营CF中。上述V1加入是为了还原完整现金图景，但OCF本身不含卖矿现金。修正计算：

```
W = 真实现金收入 - OCF + 非现金调整项
  ≈ 1,157.80 - 217.40 + 非现金调整
```

由于非现金调整项（折旧、减值等）已包含在OCF间接法计算中，更简洁的做法是直接使用OCF作为经营现金净流入的基准。
