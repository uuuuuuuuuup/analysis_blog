### 模块一：资本消耗强度

**评级：高**

顺丰控股属于重资产物流企业，资本消耗强度高：
- 2025FY资本开支97亿元，占营收3.1%，占OCF 35.1%
- 固定资产+在建工程占总资产~25%
- 拥有中国最大货运机队（顺丰航空）+ 鄂州航空枢纽
- 租赁负债约95亿元（飞机、仓库等经营租赁资本化）
- 持续投入科技（智能分拣、数智化平台）

**对穿透回报的影响**：高资本消耗意味着维持性资本开支占比较高，自由现金流与净利润的比值（FCF/NI = 179/111 = 161%）虽看似充裕，但需关注未来资本开支的持续性。

### 模块二：收款模式

**评级：应收为主，但质量尚可**

- 应收账款/营收比：9.73%（2023）→ 9.74%（2024）→ 9.93%（2025），呈微升趋势
- 应收账款周转天数：33-35天，相对稳定
- 收款比率（真实现金收入/营业收入）≈ 99.06%，接近1.0
- 客户结构：B端客户为主（企业合同物流、电商商家），账期相对规范
- ⚠️ 鹰眼预警：应收账款/营收比值持续增长，需关注回款风险

**对穿透回报的影响**：收款模式总体健康，但应收账款持续增长需警惕。收款比率>99%说明现金回收质量良好。

## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 111.17 亿元（2025FY GAAP 归母净利润）

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）= 60 亿元
E（资本开支）= 97 亿元
G（维持性资本开支系数）= 0.7（重资产物流企业，维持性资本开支占D&A的70%）
H = D × G = 60 × 0.7 = 42 亿元
I = C + D - H = 111.17 + 60 - 42 = 129.17 亿元
```

> 注：G=0.7的假设基于顺丰当前仍处于扩张期（鄂州枢纽产能爬坡、国际业务拓展），约30%的资本开支为成长性投入

### 步骤 3：分配能力验证（一票否决）

```
OCF = 276 亿元 > I = 129.17 亿元 ✅
FCF = 179 亿元 > I = 129.17 亿元 ✅
```

分配能力验证通过，不存在一票否决。

### 步骤 4：现金上游障碍

- 顺丰为A股上市公司，现金分配无上游障碍
- 少数股东损益仅~0.5亿元，占比极低
- 嘉里物流（港股子公司）分红回流可能受港股通规则影响，但影响有限

**现金上游障碍：无**

### 步骤 5：分配意愿

- 2024-2025年分红率稳定在40%
- 2025年回购方案翻倍至30-60亿元，用途变更为注销
- H股回购5亿港元
- 综合股东回报比例约47.8%

**分配意愿：强**（较2023年15%大幅改善）

### 步骤 6：可预测性

- 营收增速稳定（6-10%）
- 净利润增速波动较大（1-29%）
- OCF波动较大（受营运资本影响）
- 扣非净利润增速放缓至1.29%

**可预测性：中等**

### 步骤 7：税务泄漏修正

- A股持股>1年：股息税率为0%
- Q = 0%（假设长期持有）

### 步骤 8：粗算穿透回报率

```
R = (I × M × (1-Q%) + O) / 当前市值
  = (129.17 × 40% × 100% + 20) / 1,897
  = (51.67 + 20) / 1,897
  = 71.67 / 1,897
  = 3.78%
```

### 粗算否决门判断

```
II = 3.78%（A股门槛）
R = 3.78%

R = II → 恰好达标（边际达标）
```

**粗算结论：边际达标**（R恰好等于门槛II，继续进入因子3精算）

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1：真实现金收入还原

```
S（营业收入）= 3,082.27 亿元
T（应收账款净变动）= 306.07 - 277.15 = +28.92 亿元（增加为正，扣除）
U（预收款/合同负债净变动）= 未单独披露，按0计

保守处理：
- 应收账款净增加28.92亿元 → 扣除（收入未收到现金）
- 预收款净变动 → 未披露，不加回

真实现金收入 = S - T净增加 = 3,082.27 - 28.92 = 3,053.35 亿元
收款比率 = 3,053.35 / 3,082.27 = 99.06%
```
