## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 122.43 亿元（2024年报归母净利润）

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）≈ 6.00 亿元
G（维持性资本开支系数）= 0.4（白酒行业维持性Capex占D&A比例较低）
H = D × G = 6.00 × 0.4 = 2.40 亿元
I（Owner Earnings）= C + D - H = 122.43 + 6.00 - 2.40 = 126.03 亿元
```

### 步骤 3：分配能力验证

- OCF（121.72亿元）> 归母净利润（122.43亿元）的95% → ✅ 利润有现金支撑
- 合同负债58.72亿元，预收模式确认 → ✅ 无现金上游障碍
- **分配能力验证：通过**

### 步骤 4：现金上游障碍

- 控股股东汾酒集团，无跨境障碍（A股直接持有）
- 少数股东权益占比极低（498/39,278 = 1.27%）
- **现金上游障碍：无**

### 步骤 5：分配意愿

- 2024年支付率60.39%，较此前50%大幅提升
- 中期分红（三季报分红）常态化
- **分配意愿：强且持续提升**

### 步骤 6：可预测性

- 白酒行业需求相对稳定
- 清香型龙头地位稳固
- 2025Q1-Q3营收+5%，净利润+0.78%（增速放缓但仍正增长）
- **可预测性：中-高**

### 步骤 7：税务泄漏修正

- A股持股>1年免征股息税
- 假设长期持有，Q = 0%

### 步骤 8：粗算穿透回报率

```
R = [I × M × (1-Q%) + O] / 当前市值
R = [126.03 × 60% × 100% + 0] / 1,702
R = 75.62 / 1,702
R = 4.44%
```

**粗算否决门检查：**
- R（4.44%）> Rf（1.78%）→ ✅ 通过
- R（4.44%）> II × 0.5（1.89%）→ ✅ 通过
- R（4.44%）> II（3.78%）→ ✅ 通过

**因子 2 结论：粗算穿透回报率 R = 4.44%，通过粗算门槛**

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1：真实现金收入还原

```
S（营业收入）= 36,011 百万元
T（应收账款净变动）≈ 0（应收仅0.08百万元，可忽略）
U（合同负债净变动）≈ +372 百万元（估算：2024年末5,872 vs 2023年末~5,500）

山西汾酒为白酒"先款后货"模式，合同负债属高确定性预收，可将净增加加回：
真实现金收入 = S - T净增加 + U净增加（高确定性预收加回）
= 36,011 - 0 + 372 = 36,383 百万元

收款比率 = 36,383 / 36,011 = 101.03%
```
