四、MD&A 深度信息#

来源：AI-Tools MCP GetFinancialReport + WebSearch公开信息（深交所公告、券商研报、机构调研）

4.1 经营回顾#

2024年度经营成果：

  • 营业收入22.57亿元，同比+22.18%
  • 归母净利润3.76亿元，同比+14.32%
  • 扣非净利润3.41亿元，同比+16.91%
  • 经营活动现金流净额3.51元/股

分渠道表现：

  • 经销模式收入19.27亿元，同比+22.58%，贡献85%收入
  • 电商模式收入2.39亿元，增速放缓
  • 海外与零食量贩渠道表现亮眼
  • 2024Q4启动出海业务，布局东南亚市场调研及团队搭建

2025年一季度经营情况：

  • 营业收入5.04亿元，同比-13.99%
  • 归母净利润5,276万元，同比-42.21%
  • 扣非净利润4,582万元，同比-45.14%
  • 毛利率34.32%，净利率10.47%

2025年上半年经营情况：

  • 营业收入9.45亿元，同比-9.34%
  • 归母净利润7,460万元，同比-55.33%
  • 海外收入暴增162%

4.2 前瞻指引#

来源：WebSearch-甘源食品机构调研（127家机构调研）

2025年战略聚焦：

  1. 品牌升级：新品联动、明星代言、文旅营销
  2. 线上社交媒体互动：加强与抖音等平台合作
  3. 线下网点覆盖：深化全渠道布局
  4. 海外拓展：深化国内与东南亚市场优质渠道资源的战略协同
  5. 电商业务：拟上线风味小吃、风味坚果等线上专属产品，与京东、天猫等平台开启深度合作
  6. 传统商超：匹配直营销售人员，聚焦核心系统

盈利端展望：

  • 近期棕榈油略有下行，青豌豆已前瞻储备
  • 品宣投放、人员招聘等费用集中于Q1
  • 预期年内盈利有望逐步修复

4.3 资本配置意图#

  • 使用闲置募集资金进行现金管理（银行理财产品），授权额度9亿元
  • 使用闲置自有资金进行现金管理，安全性高、满足保本要求、流动性好、单项投资期限最长不超过12个月的投资产品，授权额度3.5亿元
  • 在建工程持续投入（2025Q3: 1.44亿元），主要为产能扩建项目
  • 回购专用证券账户持有1,605,872股

4.4 风险因素#

  1. 需求平淡风险：2025Q1营收同比下降13.99%，渠道调整影响短期业绩
  2. 应收账款增长风险：应收账款/营收比持续增长（1.49%→1.56%→4.61%）
  3. 费用投放压力：销售费用增长34.67%，主要用于薪酬福利与促销推广
  4. 原材料价格波动：棕榈油等主要原材料价格波动
  5. 海外业务不确定性：出海业务处于初期阶段

5.2 关键审计事项#

来源：WebSearch-证券之星

关键审计事项一：收入确认

  • 收入确认政策的适当性及收入确认时点的准确性

关键审计事项二：存货可变现净值

  • 存货采用成本与可变现净值孰低计量
  • 管理层按照估计售价减去至完工时估计将要发生的成本、估计的销售费用和相关税费后的金额确定可变现净值
  • 由于存货金额重大，且确定存货可变现净值涉及重大管理层判断，将存货可变现净值确定为关键审计事项
  • 审计应对：了解与存货可变现净值相关的关键内部控制，评价这些控制的设计，确定其是否得到执行，并测试相关内部控制
