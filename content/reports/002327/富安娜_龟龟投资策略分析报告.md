### 步骤 8：粗算穿透回报率

**基准情景（2024年度数据）：**

```
R = [I × M × (1 - Q%) + O] / 市值
  = [5.74 × 0.90 × 1 + 0.40] / 59.10
  = [5.17 + 0.40] / 59.10
  = 5.57 / 59.10
  = 9.42%
```

**当前情景（TTM数据，含2025Q1-Q3）：**

TTM归母净利润 ≈ 2025Q1-Q3累计1.60亿 + 2024Q4约2.49亿 = 4.09亿元

```
I_TTM ≈ 4.09 + 0.80 - 0.48 = 4.41 亿元
R_TTM = [4.41 × 0.90 + 0.40] / 59.10
      = [3.97 + 0.40] / 59.10
      = 4.37 / 59.10
      = 7.39%
```

**悲观情景（2025年全年年化）：**

2025年全年归母净利润估算 ≈ 2.50 亿元（基于Q1-Q3趋势外推）

```
I_悲观 ≈ 2.50 + 0.80 - 0.48 = 2.82 亿元
R_悲观 = [2.82 × 0.90 + 0.40] / 59.10
       = [2.54 + 0.40] / 59.10
       = 2.94 / 59.10
       = 4.97%
```
