## 因子 2：穿透回报率粗算（Top-Down）

### 步骤1：归母净利润

C = 13.71 亿元（2024A GAAP归母净利润）

⚠️ 注意：2024年归母净利润含出售民生证券股权产生的约2.6亿投资收益，扣非归母净利润为10.93亿元。

### 步骤2：Owner Earnings 粗算

```
D = 3.50 亿元（折旧摊销估算）
E = 4.00 亿元（资本开支估算）
G = 0.70（维持性资本开支系数，制造业典型值）
H = D × G = 3.50 × 0.70 = 2.45 亿元
I = C + D - H = 13.71 + 3.50 - 2.45 = 14.76 亿元
```

**Owner Earnings = 14.76 亿元**

### 步骤3：分配能力验证

- 2024A OCF = 13.45 亿元 > 0 ✅
- Owner Earnings = 14.76 亿元 > 0 ✅
- ⚠️ 但2025Q1-3 OCF = -4.05 亿元，现金流趋势恶化

### 步骤4：现金上游障碍

- A股直接持有，无跨境资金障碍
- **现金上游障碍：无**

### 步骤5：分配意愿

- 上市以来累计分红约64亿元
- DPS已提升（每10股从7元提升至10元）
- **分配意愿：强**

### 步骤6：可预测性

- 营收受地产周期影响，连续两年下降
- 2024年扣非净利-3.67%，2025Q1-3扣非净利-17.56%
- 非经常性损益波动大（2024年+2.78亿，2025Q1-3 -0.39亿）
- **可预测性：中等偏低**

### 步骤7：税务泄漏修正

- A股持股>1年：实际税负5%
- Q = 5%

### 步骤8：粗算穿透回报率

```
M = 67%（支付率锚定值）
O = 0 亿元（无持续回购）

```

**粗算结论：✅ 通过**

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤1：真实现金收入还原

```
S = 104.94 亿元（2024A营业收入）

- 应收账款净增加 → 扣除
```
