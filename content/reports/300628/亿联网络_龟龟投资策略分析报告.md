## 三、因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤1：归母净利润**
C = 26.48 亿元

**步骤2：Owner Earnings 粗算**
- D（折旧摊销）= 1.10 亿元
- G（维持性资本开支系数）= 0.7（轻资产模型，维持性capex低于D&A）
- H = D × G = 1.10 × 0.7 = 0.77 亿元
- I = C + D - H = 26.48 + 1.10 - 0.77 = 26.81 亿元

**步骤3：分配能力验证**
- OCF = 25.09 亿元
- OCF/净利润 = 94.8% → 高质量
- OCF - 年度分红 = 25.09 - 23.99 = 1.10 亿元 → 正向结余
- **✅ 分配能力验证通过**

**步骤4：现金上游障碍**
- 非控股公司型结构，主要运营实体在国内
- 海外子公司利润回流无重大障碍
- **现金上游障碍：低**

**步骤5：分配意愿**
- 2024年度支付率90.5%（含中期分红）
- 上市以来累计派现超70亿元（IPO仅募资16亿元）
- DPS持续提升：2023年10派9元→2024年10派13元
- **分配意愿：强**

**步骤6：可预测性**
- SIP话机全球第一，市场份额稳定
- 净利率稳定在45-50%区间
- 收入增速有波动（2023年-9.61%，2024年+29.28%）
- **可预测性：中等偏高**

**步骤7：税务泄漏修正**
- A股持股>1年：股息免税，Q% = 0%
- 采用长期投资假设：Q% = 0%

**步骤8：粗算穿透回报率**
```
R = [I × M × (1 - Q%) + O] / 当前市值
  = [26.81 × 0.85 × 1.0 + 0.10] / 445.57
  = [22.79 + 0.10] / 445.57
  = 22.89 / 445.57
  = 5.14%
```

**粗算否决门检查：**
- R = 5.14% > Rf = 1.78% → ✅
- R = 5.14% > II × 0.5 = 1.89% → ✅
- R = 5.14% > II = 3.78% → ✅

**粗算结论：✅ 通过**
