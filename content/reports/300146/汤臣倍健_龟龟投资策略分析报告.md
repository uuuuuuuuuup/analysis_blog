## 因子 2：穿透回报率粗算（Top-Down）

### 步骤 1：归母净利润

C = 7.82 亿元

### 步骤 2：Owner Earnings 粗算

```
D（折旧摊销）= 3.50 亿元
G（维持性资本开支系数）= 0.6
H = D × G = 3.50 × 0.6 = 2.10 亿元
I（Owner Earnings）= C + D − H = 7.82 + 3.50 − 2.10 = 9.22 亿元
```

### 步骤 8：粗算穿透回报率

```
R = [C × M × (1 − Q%) + O] / 当前市值
  = [7.82 × 91.5% × (1 − 5%) + 1.00] / 189.30
  = [7.82 × 0.915 × 0.95 + 1.00] / 189.30
  = [6.797 + 1.00] / 189.30
  = 7.797 / 189.30
  = 4.12%
```

| 检查项 | 阈值 | 实际值 | 结果 |

| R vs II | R ≥ 3.78% | 4.12% > 3.78% | ✅ 通过 |

### 因子 2 结论：✅ 粗算通过

R = 4.12% > II = 3.78%，粗算穿透回报率达标。

## 因子 3：穿透回报率精算（Bottom-Up）+ 现金质量审计

### 步骤 1：真实现金收入还原

```

- 应收账款净增加 1.10 亿元 → 扣除（收入未收到现金）

真实现金收入 = S − T = 62.65 − 1.10 = 61.55 亿元
收款比率 = 61.55 / 62.65 = 98.2%
```
