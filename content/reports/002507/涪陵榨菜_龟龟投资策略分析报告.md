### 模块一：资本消耗强度

**评级：低**

- 榨菜生产为轻资产模式，主要资本开支为乌江绿色智能化生产基地（在建工程增长58.26%）
- 维持性资本开支占比低，折旧摊销仅约0.80亿元/年
- 资本消耗强度低，大部分利润可转化为自由现金流

### 模块二：收款模式

**评级：现结为主（优秀）**

- 应收账款仅29.17百万元（占总资产0.3%），周转天数8.72天
- 收款比率 = 99.14%（真实现金收入/营业收入），接近1
- 传统经销模式以先款后货为主，收款质量极高
- ⚠️ 新增直销客户导致应收增长256%，需持续关注

### 模块三：竞争格局与护城河

**护城河评级：窄偏宽**

**第一层（非技术性护城河）：**
| 维度 | 评级 | 说明 |
|---|---|---|
| 品牌优势 | ★★★★☆ | "乌江"品牌为榨菜品类代名词，消费者心智占据绝对优势 |
| 规模经济 | ★★★★☆ | 产销量行业第一，规模效应显著，成本优势明显 |
| 渠道网络 | ★★★☆☆ | 传统商超覆盖广泛，但新渠道（电商/餐饮/B端）仍在拓展 |
| 转换成本 | ★★☆☆☆ | 消费品转换成本较低 |

**第二层（技术性护城河）：**
| 维度 | 评级 | 说明 |
|---|---|---|
| 原材料壁垒 | ★★★★☆ | 青菜头主要产自涪陵及周边，地理依赖性强，公司具备产地优势 |
| 行业标准 | ★★★☆☆ | 参与制定国际、国家、行业等多项标准 |
| 配方工艺 | ★★★☆☆ | 三清三洗三腌三榨工艺，有一定技术壁垒 |

**竞争格局判断**：包装榨菜市场CR1超30%，涪陵榨菜绝对龙头地位稳固。但行业进入存量竞争，海天、李锦记等跨界竞争者切入赛道，同质化竞争加剧。护城河整体"窄偏宽"，品牌+原料壁垒是核心优势。

### 模块四：周期性

**评级：弱周期**

- 榨菜为必需消费品，需求刚性较强
- 但行业已步入成熟期，增速降至个位数
- 原材料（青菜头）价格受气候影响存在波动，但公司可通过提价转嫁
- 非强周期股，周期性修正不适用

### 模块五：人力资本依赖

**评级：低**

- 生产工艺标准化程度高，不依赖核心技术人员
- 人力投入回报率203-229%，人均产出高
- 管理层变动风险：2026年2月总经理变更（高翔→夏强伟），但国企人事变动属正常

### 模块六：管理层与治理结构

**评级：一般**

## 【因子 2：穿透回报率粗算（Top-Down）】

### 步骤 1-2：归母净利润与 Owner Earnings

```
C（归母净利润）= 7.68 亿元
D（折旧摊销）≈ 0.80 亿元
G（维持性资本开支系数）= 0.40（消费品行业典型值）
H = D × G = 0.80 × 0.40 = 0.32 亿元
I（Owner Earnings）= C + D - H = 7.68 + 0.80 - 0.32 = 8.16 亿元
```

### 步骤 3：分配能力验证

- OCF = 6.57 亿元 > 股息支付 4.85 亿元 → ✅ 分配能力充足
- FCF ≈ 4.07 亿元 < 股息支付 4.85 亿元 → ⚠️ FCF不足以完全覆盖股息，但广义现金充裕可弥补

### 步骤 4：现金上游障碍

- 涪陵榨菜为A股上市公司，无港股通/跨境分红障碍
- 少数股东权益极小，利润几乎全部归属母公司
- **现金上游障碍：无**

### 步骤 5：分配意愿

- FY2025支付率63.1%，较FY2024的~52%显著提升
- 公司明确表示"侧重通过现金分红回报股东"
- 无回购计划，但分红力度持续加大
- **分配意愿：强**

### 步骤 6：可预测性

- 营收5年CV（变异系数）≈ 3.1%，波动极低
- 净利润5年CV ≈ 8.5%，波动较低
- 毛利率稳定在50-54%区间
- **可预测性：高**

### 步骤 7：税务泄漏修正

- A股长期持有（>1年）股息免税 → Q = 0%
- 无税务泄漏

### 步骤 8：粗算穿透回报率

```
R = [C × M × (1 − Q%) + O] / 当前市值
  = [7.68 × 0.631 × 1.0 + 0] / 143.20
  = 4.846 / 143.20
  = 3.38%
```

### 粗算否决门判断

```
II = 3.78%
R = 3.38%
II × 0.5 = 1.89% ≤ R = 3.38% < II = 3.78%
→ 标注"边际不达标"，继续进入因子 3
```

**粗算结论：⚠️ 边际不达标（R = 3.38% < II = 3.78%，差距0.40pct）**

### 步骤 1：真实现金收入还原

```

真实现金收入 = S - T = 2,431.92 - 20.98 = 2,410.94 百万元
收款比率 = 2,410.94 / 2,431.92 = 99.14%

**收款比率99.14%，接近1，收款质量极高。**

- 新增3名直销客户导致应收增长，信用减值损失激增1,387.24%（基数极小）

### 步骤 6：准则差异核查

- **A股收入确认激进度**：公司以经销为主，先款后货模式，收入确认保守

### 步骤 7：真实可支配现金结余

```

#### 合理性校验

```
AA_fcf = OCF - Capex（直接来自现金流量表）

FY2025: AA_fcf = 657 - 250 = 407 百万元

AA / AA_fcf = 456 / 407 = 1.12 < 1.5 → ✅ 校验通过

FY2023-FY2025平均:

AA_avg = 429 百万元
AA_avg / AA_fcf_avg = 429 / 403 = 1.06 < 1.5 → ✅ 校验通过
```
