模块一：资本消耗强度#

评级：中低

  • 2024FY Capex/Revenue ≈ 1.60/24.12 = 6.6%（中等偏低）
  • 2025年Capex显著加大（固定资产从4.28亿增至5.98亿），主要用于产能扩建
  • 公司处于扩张期，资本消耗有所增加但仍可控
  • 轻资产特征明显：固定资产/总资产 = 427.73/2,167.04 = 19.7%

模块二：收款模式#

评级：现结为主，边际恶化

  • 2024FY 应收账款周转天数仅2.16天，收款极快
  • 但2025Q3恶化至5.01天，应收账款从2,032万增至4,681万（+130%）
  • 恶化原因：直营模式扩张（KA商超、线上）带来账期延长
  • 经销商渠道占比~75%，以先款后货为主
  • 整体收款模式仍优于多数消费品企业，但趋势需警惕

模块三：竞争格局与护城河#

护城河评级：窄

第一层（非技术性）：

  • 品牌认知：“劲仔"在鱼类零食细分领域有较高知名度，“鱼类零食第一股”
  • 渠道网络：覆盖全国经销网络，但渠道控制力有限
  • 规模优势：鱼制品市场份额~5.6%，鹌鹑蛋品类市占率13%+
  • 但：行业CR5<20%，格局极分散，进入门槛低

第二层（技术性）：

  • 配方/工艺：有一定技术积累，但食品行业技术壁垒天然较低
  • 专利保护：拥有多项发明专利，但可替代性强
  • 客户转换成本：低，消费者品牌忠诚度有限

护城河趋势：中性偏负面 — 竞争加剧，品牌推广费大幅增长（+67.66%），但尚未形成强护城河

模块四：周期性#

评级：弱周期

  • 休闲零食属日常消费品，需求相对稳定
  • 但公司产品（鱼制品、鹌鹑蛋）偏向可选消费，经济下行期有一定影响
  • 原材料（鱼类）价格波动带来成本周期性
  • 行业仍在增长期（CAGR 11.6%），非结构性衰退

模块五：人力资本依赖#

评级：中

  • 生产端：自动化程度提升中，但仍需大量一线工人
  • 销售端：经销商管理需人力，但非关键人物依赖
  • 管理层：创始人周劲松为核心，持股38.22%，深度绑定
  • 整体：人力依赖中等，不存在关键人物风险

模块六：管理层与治理结构#

评级：良好

  • 创始人周劲松任董事长兼总经理，持股38.22%，利益高度一致
  • 夫妻合计控制46.25%，控制权稳定
  • 分红慷慨：连续高分红，2024年支付率61.5%，含中期分红
  • 回购积极：2025年回购约1亿元
  • 关注点：
    • 2025年盈利下滑但仍在加大品牌投入，需观察投入产出比
    • 应收账款快速增长，需关注渠道风控
    • 资产负债率从12.8%升至39.1%，杠杆率上升较快

模块七：监管与政策风险#

评级：低

  • 食品安全监管：行业固有风险，但公司未发生重大食品安全事件
  • 环保监管：食品加工环保要求适中，公司合规经营
  • 税收政策：高新技术企业税收优惠（如有）需关注持续性

模块八：MD&A 解读#

前瞻：中性偏负面

  • 2025年前三季度增收不增利，净利润同比-20.21%
  • 原材料成本上升+品牌推广费大幅增加挤压利润
  • 公司提出"三年倍增计划”，但短期盈利承压
  • 机构预期2025全年净利润约2.5亿元（同比-14%），2026年恢复增长至3.2亿元
  • 管理层对长期增长保持信心，但短期面临阵痛期

因子 2：穿透回报率粗算（Top-Down）

计算步骤#

步骤 1：归母净利润

  • C = 2.9133 亿元（2024FY）

步骤 2：Owner Earnings 粗算

  • D（折旧摊销）≈ 0.85 亿元
  • G（维持性资本开支系数）= 0.6（食品制造业中等水平）
  • H = D × G = 0.85 × 0.6 = 0.51 亿元
  • I = C + D - H = 2.9133 + 0.85 - 0.51 = 3.2533 亿元

步骤 3：分配能力验证

  • OCF = 5.42 亿元 > I = 3.25 亿元 → ✅ 现金流足以支撑分配

  • A股直接持有，无跨境分红障碍 → 无障碍

步骤 5：分配意愿

  • 近3年支付率序列：[78.6%, 63.5%, 61.5%]
  • 2024年首次推出中期分红
  • 2025年回购约1亿元
  • 分配意愿：强

步骤 6：可预测性

  • 收入5年CAGR约28%，但2025增速骤降至~2%
  • 盈利波动性增大，2025年净利润预计下滑14%

─── 主轨：市值口径 ───
GG = [AA × M × (1-Q%) + O] / 当前市值
   = [3.4682 × 0.62 × 0.95 + 0.50] / 46.63
   = [2.0424 + 0.50] / 46.63
   = 2.5424 / 46.63
   = 5.45%

─── 辅轨：EV口径 ───
