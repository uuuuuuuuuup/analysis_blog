决策理由#

  1. 穿透回报率勉强达标：精算GG=4.32%仅高于门槛3.78%约0.54pct，安全边际薄弱
  2. 控股公司折价：约33%的账面利润为未实现权益法收益，实际可分配现金低于账面利润
  3. 短期业绩承压：改扩建工程导致通行费收入下降，OCF/FCF连续两年下滑
  4. 净负债状态：有息负债555亿元 vs 现金101亿元，广义净现金为-454亿元，无现金保护
  5. 正面因素：护城河极宽（特许经营垄断）、收款模式优秀、分红意愿强、央企治理规范

情景分析：改扩建完成后的潜在回报#

若改扩建完成后FCF恢复至52亿元（2024年水平）：

  • GG = [52.0 × 0.5498 + 1.55] / 621.8 = 30.14 / 621.8 = 4.85%
  • 安全边际 = 4.85 - 3.78 = 1.07 pct
  • 目标买入价 = 621.8 × (4.85/3.78) / 67.95 × 0.70 = 8.27 元
  • 距离当前价 = (8.27-9.15)/9.15 = -9.6% → 中优先级观察
⚠️ 风险提示
    1. 改扩建延期风险：京津塘高速改扩建若延期，通行费收入恢复将推迟，FCF持续承压
    2. 政策风险：高速公路免费通行天数增加、通行费下调等政策可能侵蚀收入
    3. 投资收益波动：联营公司业绩波动导致投资收益不稳定，影响利润可预测性
    4. 高杠杆风险：有息负债555亿元，有息资产负债率33.5%，利率上行将增加财务成本
    5. 控股公司结构风险：大量权益法投资收益未实现为现金，利润质量低于纯运营型公司
📋 后续跟踪事项
    1. 跟踪改扩建进度：京津塘高速改扩建完工时间及车流量恢复情况
    2. 跟踪FCF恢复：关注季度OCF和Capex数据，验证FCF是否回升至50亿元以上
    3. 跟踪股息政策：关注分红比例是否维持在50%+，以及是否有中期分红
    4. 跟踪股价位置：若股价回落至7.3-8.3元区间，安全边际改善，可提升优先级
    5. 跟踪REITs运作：关注后续REITs发行计划，轻资产化进展

因子 2：穿透回报率粗算（Top-Down）

计算步骤#

📊 展开计算过程

步骤 1：归母净利润 C = 46.10 亿元

步骤 2：Owner Earnings 粗算 D（折旧摊销）≈ 30.00 亿元 G（维持性资本开支系数）= 0.35（高速公路维护开支占比低） H = D × G = 30.00 × 0.35 = 10.50 亿元 I = C + D - H = 46.10 + 30.00 - 10.50 = 65.60 亿元

步骤 3：分配能力验证 OCF（2025）= 65.10 亿元 > 股息支付 25.35 亿元 ✅ FCF（2025）≈ 45.10 亿元 > 股息支付 25.35 亿元 ✅ → 分配能力验证通过

步骤 4：现金上游障碍 A股直接持有，无跨境障碍 ✅

步骤 5：分配意愿 支付率均值 ≈ 54%，连续高分红，2024-2025年实施回购 → 分配意愿强

步骤 6：可预测性 高速公路通行费收入可预测性高，但投资收益波动带来不确定性 → 可预测性中

步骤 7：税务泄漏修正 A股持股>1年：Q = 0%（免税）

步骤 8：粗算穿透回报率 R = [I × M × (1-Q%) + O] / 市值 = [65.60 × 0.5498 × 1 + 1.55] / 621.8 = [36.07 + 1.55] / 621.8 = 37.62 / 621.8 = 6.05%

粗算否决门判断#

粗算否决门判断
条件 结果
R < Rf (1.78%)? 否 ✅
R < II×0.5 (1.89%)? 否 ✅
II×0.5 ≤ R < II? 否（R=6.05% > II=3.78%）✅
粗算结论
✅ 通过 — R=6.05%显著高于门槛II=3.78%
