四、MD&A 深度信息#

数据来源：mcp_AI_Tools_GetFinancialReport + WebSearch

4.1 经营回顾#

2024年度经营回顾：

  • 2024年实现营业收入235.78亿元，同比增长16.41%
  • 归母净利润55.17亿元，同比增长20.22%
  • 经营活动现金流量净额47.28亿元，同比增长5.15%
  • 白酒销售228.65亿元，占营业收入96.98%
  • 销售费用61.82亿元，销售费用率26.22%，同比下降0.6个百分点
  • 营业成本率20.10%，较去年同期下降0.83个百分点
  • 来源：WebSearch - 2024年度财务决算报告

2025年前三季度经营回顾：

  • 1-3Q25实现收入164.25亿元，同比-13.9%
  • 归母净利润39.60亿元，同比-16.6%
  • 3Q25单季收入25.45亿元，同比-51.7%；净利润3.00亿元，同比-74.6%
  • 三季度收入与业绩大幅调整，原因：一是需求环境持续承压，影响白酒实际动销；二是公司为保障渠道健康，选择主动调整，缓解渠道压力
  • Q3末合同负债13.44亿元，环比降0.84亿元，同比下滑约61.76%
  • 来源：WebSearch - 国泰海通证券研报 2025-11-05

4.2 前瞻指引#

  • 2025年不再提业绩增速目标，总经理周庆伍表示"降速发展成为行业共识"
  • 此前公司原定2025年实现300亿营收目标，现已不再强调
  • 机构预期2025年净利润约43-45亿元，同比下滑约19%
  • 来源：WebSearch - 每日经济新闻 2025-04-27

4.3 资本配置意图#

  • 智能园（酿酒生产智能化技术改造项目）持续推进，在建工程2024年末约10.39亿元
  • 2025年前三季度投资现金流净额-19.86亿元，主要用于产能建设
  • 公司使用闲置自有资金进行委托理财（2025年4月公告）
  • 前三季度将49.31亿元资金配置于国债逆回购等低风险金融产品
  • 来源：WebSearch - 新浪财经 2025-10-31

4.4 风险因素#

  1. 市场风险：消费需求不足、居民消费降级风险，白酒行业竞争激烈
  2. 库存风险：渠道去库存压力，经销商订货意愿低迷，合同负债大幅回落
  3. 省外扩张风险：华北、华南市场增速放缓，竞争加剧
  4. 行业竞争风险：次高端赛道（汾酒、舍得等）竞争加剧
  5. 食品安全风险：行业共性风险
  6. 来源：WebSearch - 新浪财经 2025-04-28
