模块一：资本消耗强度#

评估：低-中

  • 2024FY Capex/Revenue = 22.23 / 1,157.80 = 1.92% → 资本消耗强度低
  • 2024FY Capex/OCF = 22.23 / 217.40 = 10.22% → 经营现金流可轻松覆盖资本开支
  • 固定资产/总资产 = 346.19 / 1,537.18 = 22.52% → 重资产比例中等
  • 乳制品行业属于轻-中等资本消耗行业，维持性资本开支需求不高

模块二：收款模式#

评估：预收/现结为主（优秀）

  • 应收账款周转天数：9.3天（2024FY）→ 8.99天（2025Q3）→ 极快
  • 应收账款/营业收入 = 28.78 / 1,157.80 = 2.49% → 极低
  • 收款比率 ≈ 100% → 现金回收效率极高
  • 对下游经销商议价能力强，基本实现"现款现货"
  • 应付账款周转天数：61.6天 → 对上游供应商有较强账期优势
  • 现金循环周期：2.57天（2024FY）→ -1.68天（2025Q3）→ 已转为负值，运营效率卓越
