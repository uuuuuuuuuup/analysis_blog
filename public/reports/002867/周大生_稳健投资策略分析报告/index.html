模块一：资本消耗强度#

评级：低
  • 2024FY Capex/Revenue = 205.3/13,891.3 = 1.48%，极低
  • 2024FY Capex/OCF = 205.3/1,855.9 = 11.07%，低
  • 珠宝零售为轻资产模式，主要资本开支为门店装修和系统建设
  • 在建工程806.4百万元（2024FY），主要为总部大楼建设，属一次性投入
  • 维持性资本开支系数 G ≈ 0.3-0.5

模块二：收款模式#

评级：现结为主，少量应收
  • 应收账款周转天数：23.2天（2024FY），13.4天（2025Q3）
  • 收款比率 ≈ 100%（应收账款净变动极小）
  • 加盟模式以预收/现结为主，自营门店为现销
  • 应收账款主要来自加盟商赊销，公司已主动降低赊销额度
  • 坏账风险低：2024年资产减值损失-33.6百万元，占应收比例小

模块三：竞争格局与护城河#

非技术性护城河：窄偏宽

  • 品牌壁垒：品牌价值连续多年内地珠宝品牌第一
  • 渠道壁垒：4,831家门店构成的网络效应，新进入者难以短期复制
  • IP合作：“周大生×国家宝藏"等文化IP构建差异化

技术性护城河：窄

  • 黄金珠宝行业产品同质化程度高，技术壁垒有限
  • 品牌溢价主要来自营销和渠道，非技术差异化
  • 镶嵌类产品有一定设计壁垒，但占收入比仅~4%
综合护城河评级：窄
  • 品牌和渠道提供一定护城河，但行业竞争激烈
  • 金价透明度高，产品差异化有限
  • 头部品牌（周大福、老凤祥）份额更大

模块四：周期性#

评级：弱周期
  • 黄金珠宝消费受金价波动影响明显：2024年金价高企压制终端需求，营收-14.73%
  • 2025Q3营收进一步下降37.35%，但净利率改善
  • 金价下跌时，消费者买涨不买跌的心理也会抑制需求
  • 长期看，婚庆需求和保值需求提供一定稳定性
  • 非强周期行业，但金价波动带来营收波动

模块五：人力资本依赖#

评级：低
  • 珠宝零售为标准化运营，对核心人才依赖度低
  • 加盟模式进一步降低了对直营人员的依赖
  • 管理层稳定：周宗文自公司成立起担任董事长

模块六：管理层与治理结构#

  • 审计机构为安永华明（四大），治理规范
  • 潜在风险：家族企业可能存在关联交易不透明

模块七：监管与政策风险#

步骤 2：Owner Earnings 粗算

⚠️ 此值高于OCF口径FCF，需进行合理性校验

合理性校验：
AA_fcf = OCF - Capex = 1,855.9 - 205.3 = 1,650.6 百万元

比较：AA = 2,351.74 vs AA_fcf = 1,650.6
AA / AA_fcf = 1.42 → AA > AA_fcf × 1.0 但 < AA_fcf × 1.5

→ 校验通过（AA ≤ AA_fcf × 1.5），但偏高
→ 原因：W的计算可能未包含所有经营支出项（如存货采购的现金支出可能被低估）
→ 采用保守修正：AA_conservative = min(AA, AA_fcf × 1.2) = min(2,351.74, 1,980.72)
→ AA = 1,980.72 百万元 ≈ 19.81 亿元

但考虑到2025年经营恶化，需进行跨年平滑：

2024FY FCF = 1,650.6 百万元
