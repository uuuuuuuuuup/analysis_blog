## 因子 2：穿透回报率粗算（Top-Down）

### 计算步骤

**步骤1：归母净利润**
- C = 51.05 亿元

**步骤2：Owner Earnings 粗算**
- D（折旧摊销）= 10.0 亿元（估）
- G（维持性资本开支系数）= 0.85（成熟制造业，大部分capex为维持性）
- H = D × G = 10.0 × 0.85 = 8.50 亿元
- I（Owner Earnings）= C + D - H = 51.05 + 10.0 - 8.50 = **52.55 亿元**

**步骤3：分配能力验证**
- OCF FY2025 ≈ 73.53 亿元 > I × M = 52.55 × 0.975 = 51.24 亿元 → ✅ 通过
- OCF足以覆盖分配需求

**步骤4：现金上游障碍**
- 全部境内子公司，无跨境障碍 → 障碍 = 0

**步骤5：分配意愿**
- 连续26年分红，近年支付率97-98% → **极强**
- DPS已提升（10派14.5元 vs 上年10派14.1元）

**步骤6：可预测性**
- 营收5年CV = 8.4%（低波动）→ 可预测性较高
- 但OCF波动较大（FY2023仅33.88亿）→ 现金流可预测性中等

**步骤7：税务泄漏修正**
- A股长期持有（>1年）股息免税 → Q = 0%

**步骤8：粗算穿透回报率**

```
R = (I × M × (1-Q%) + O) / 当前市值
R = (52.55 × 0.975 × 1.0 + 0) / 996.86
R = 51.24 / 996.86
R = 5.14%
```

### 粗算否决门判断

| 条件 | 结果 | 判断 |
|---|---|---|
| R < Rf (1.78%)? | 5.14% > 1.78% | ✅ 通过 |
| R < II×0.5 (1.89%)? | 5.14% > 1.89% | ✅ 通过 |
| R < II (3.78%)? | 5.14% > 3.78% | ✅ 通过 |

**粗算结论：✅ 通过，R = 5.14% > II = 3.78%，继续进入因子3精算**

```

```
